Privacy Policy

Effective Date: December 2025

1. Data Collection

We collect the minimum amount of data required for the bot's functionality:

  • Discord User ID: Used to associate scores, inventory, and rank with your account.
  • Server Settings: We store the Channel ID configured for trivia output.
  • Temporary Data: Display Names and Avatars are accessed briefly to generate images but are not permanently stored on our servers.

2. Data Usage

Your data is used solely to:

  • Maintain global leaderboards and user profiles.
  • Track inventory ownership and auction history.
  • Calculate game statistics (e.g., accuracy streaks).

3. Third-Party Services

Trivia content is generated via OpenRouter APIs. When generating a question, no personal user data is transmitted to the AI provider.

4. Data Deletion

If you wish to have your data removed from the system, please join our Support Server and submit a request. Upon deletion, all game progress and stats will be permanently lost.
